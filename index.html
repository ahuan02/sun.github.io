快速开始

三步上手

下载、安装、运行，一分钟搞定。

1️⃣

选择版本下载

推荐 Windows 用户下载 MSI 安装版，一步到位。

2️⃣

安装 / 解压

MSI EXE 双击安装，JAR 放好就行。

3️⃣

启动即用

双击 CoreTools.exe 或执行 java -jar CoreTools.jar

# Windows MSI 安装后，从开始菜单或桌面快捷方式启动

# 便携版exe双击安装后运行
> CorTools.exe

# JAR 包方式（需 JDK 26+）
> java -jar CoreTools-1.0.0.jar

# 自定义 JVM 参数（可选）
> java -Xmx2G -Dfile.encoding=UTF-8 -jar CoreTools-1.0.0.jar
